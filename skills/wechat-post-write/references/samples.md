# 文章样本使用说明

本文档说明如何参考 `samples/` 目录下的优秀公众号文章。

## 样本概览

| 文件 | 亮点特点 |
|------|----------|
| `01_OpenClaw全球首聚.md` | 事件型报道、现场感强、悬念式标题、数据驱动 |
| `02_Claude5史诗级泄露.md` | 深度分析型、引用原始来源、技术细节详尽、行业视角 |
| `03_Claude_Code配置神器.md` | 工具推荐型、使用场景清晰、实战经验分享 |
| `04_Claude_Cowork开源平替.md` | 竞品对比型、分类清晰、优缺点并述 |
| `05_9个优质GitHub项目.md` | 列表型、简洁高效、每项精炼 |

## 参考要点

### 标题风格
- **数据驱动**：`3.5万Star`、`8小时`、`80.9%`
- **悬念设置**：`太顶了`、`史诗级泄露`、`杀疯了`
- **反差对比**：`开源平替`、`堪称外挂`

### 开头写法
- 用具体数据吸引注意
- 用场景/故事引入
- 设置悬念或抛出问题

### 语气特点
- **专业但不严肃**：准确的技术描述 + 轻松的行文节奏
- **选择性引用**：只在关键信息处标注来源
- **段落适中**：长句与短句交替，不堆砌

### 结构灵活
- 根据内容选择合适的结构
- 不是所有文章都要完整的"痛点→亮点→场景→上手"
- 自然流畅比模板更重要
