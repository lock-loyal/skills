# 调试方式

通过print函数调试，打印信息会输出到运行日志中，子模型中的print信息会重定向到总模型的日志中

打印日志会记录打印的位置，如果是在子模型中打印的，会记录子模型打印时，总模型的调用子模型位置和子模型的打印位置
