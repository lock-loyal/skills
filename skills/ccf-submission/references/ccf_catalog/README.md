## 文件结构

```

├── README.md                    # 本文件

├── all_conferences.json         # 所有会议（JSON格式）
├── by_field/                    # 按领域分类的Markdown文件
│   ├── 01_architecture.md       # 计算机体系结构/并行与分布计算/存储系统
│   ├── 02_network.md            # 计算机网络

│   ├── 06_theory.md             # 计算机科学理论
│   ├── 07_graphics.md           # 计算机图形学与多媒体
│   ├── 08_ai.md                 # 人工智能
│   ├── 09_hci.md                # 人机交互与普适计算
│   └── 10_interdisciplinary.md  # 跨学科/交叉/综合/新兴领域
└── by_class/                    # 按级别分类的Markdown文件
    ├── class_a.md               # A类期刊和会议
    ├── class_b.md               # B类期刊和会议

```

## 数据格式

每个条目包含以下字段：

```json
{
  "field": "领域名称",
  "type": "期刊|会议",
  "class": "A|B|C",
  "rank": 1,
  "abbreviation": "简称",
  "full_name": "全称",
  "publisher": "出版社",
  "url": "网址"
}
```
